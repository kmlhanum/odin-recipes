Welcome to List of Nusantara Recipes

This project is a required assignment to successfully finish The Odin Project's HTML Foundation module. The central focus is on building a functional recipe website that utilizes linking between pages and demonstrates the correct implementation of ordered lists for instructions and unordered lists for ingredients.
